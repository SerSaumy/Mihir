## Flutter & Dart Packages

- **Flutter** - UI framework
- **Riverpod** - State management
- **Drift** - SQLite ORM
- **Dio** - HTTP client
- **GoRouter** - Navigation
- **Cached Network Image** - Image caching
- And many more (see `pubspec.yaml`)

## Node.js Packages

- **Axios** - HTTP client for extension bridge
- **Cheerio** - HTML parsing (JSoup replacement)
- **Node Fetch** - Fetch API implementation
- **UUID** - UUID generation
